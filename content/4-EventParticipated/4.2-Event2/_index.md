# Report on Cloud Technology & Modern Software Development Seminar

### Event Objectives

- Share advanced security solutions combining AWS WAF with Machine Learning (ML).
- Introduce multiplayer game development methods utilizing Serverless WebSockets architecture on AWS.
- Guide on building a Graph Knowledge Base with AWS Neptune to empower GraphRAG systems.
- Provide best practices regarding containerization technology (Docker) and effective team collaboration skills.
- Map out a practical career development roadmap from IT Helpdesk transitioning to Cloud-DevOps.

### Speaker List

- **Bảo Huỳnh** - Expert in Containerization Technology (Docker)
- **Lê Hoàng Gia Đại** - Expert in Cybersecurity & AI/ML at AWS
- **Nguyễn Quốc Bảo** - Cloud & Game Development Specialist
- **Trương Phước** - Team Work & Agile Coach
- **Việt Phát** - AI & Graph Database Engineer
- **Vinh Trần** - Senior Sysadmin & Cloud-DevOps Engineer

### Highlighted Contents

#### Docker - A Containerization Technology (Bảo Huỳnh)
- **Core Concept:** The essence of containerization and how Docker solves the infamous "it works on my machine but breaks on the server" problem.
- **Docker Architecture:** How to optimize Dockerfiles, manage Images, Containers, and Volumes for persistent data storage.
- **Optimizing Development Workflows:** Utilizing Docker in local development environments to synchronize settings among project team members.

#### Combining AWS WAF with Machine Learning for Cyber Attack Detection (Lê Hoàng Gia Đại)
- **Limitations of Traditional WAF:** Why static, pattern-based rulesets are no longer sufficient against today's sophisticated cyber attacks.
- **Machine Learning Integration:** Methodology of combining AWS WAF with ML models to analyze traffic behavior, detect anomalies, and block malicious requests in real time.
- **Intelligent Defense Architecture:** Designing smart, secure infrastructure perimeters while minimizing false positive alarm rates on AWS.

#### Multiplayer in the Cloud: Connecting Godot Clients with AWS WebSockets (Nguyễn Quốc Bảo)
- **Real-Time Connectivity Challenges:** Overcoming latency and networking bottlenecks when building infrastructure for multiplayer games.
- **AWS WebSockets Solution:** Leveraging Amazon API Gateway (WebSocket API) combined with AWS Lambda to manage concurrent connections from Godot Engine game clients.
- **State Synchronization:** An optimized, cost-efficient data transmission and storage model based on a Serverless architecture.

#### Effective Teamwork (Trương Phước)
- **Barriers in Teamwork:** Analyzing common roots of conflicts, misalignment, and dropping performance within tech project teams.
- **Improvement Strategies:** Core methods to build mutual trust, streamline communication channels, and break down tasks scientifically.
- **Applying Agile/Scrum Frameworks:** How adopting daily Standups and Sprint Reviews helps tech teams adapt rapidly to changing project demands.

#### AWS Neptune for Building a Graph Knowledge Base for GraphRAG (Việt Phát)
- **The Rise of GraphRAG:** Why traditional RAG built on Vector Search hits context-linking boundaries, and how GraphRAG solves this by implementing graph databases.
- **Leveraging AWS Neptune:** Designing entities (Nodes) and relationships (Edges) on a fully managed graph database to store knowledge.
- **Advanced Querying Pipeline:** Extracting graph-structured information to feed into LLMs, resulting in highly accurate, comprehensive, and context-aware AI outputs.

#### From IT Helpdesk to Senior Sysadmin: A Self-Learning Journey to Cloud-DevOps (Vinh Trần)
- **Real-World Story:** A motivational journey of stepping out of the comfort zone, moving from handling basic hardware/OS issues as an IT Helpdesk up to a Senior Sysadmin role.
- **Self-Learning Path:** Core professional certifications and key technological skills to focus on for career breakthroughs.
- **Transitioning to Cloud-DevOps:** A comprehensive skill roadmap covering Infrastructure as Code (IaC), CI/CD pipelines, automation scripting, and adopting a cloud-native mindset in enterprises.

### Key Learnings

#### Design Mindset
- **Data-Relationship First Approach:** Understanding the critical importance of data connections in Graph Databases (AWS Neptune) for complex AI applications like GraphRAG, rather than relying solely on traditional relational databases.
- **Proactive Security Mindset:** Shifting from manual security rule configurations to automated, AI-driven anomaly detection (AWS WAF + ML).
- **People-Centric View:** Realizing that technology is merely a tool; optimizing team communication and workflows is the primary leverage that drives a project to success.

#### Technical Architecture
- Mastered establishing bi-directional communication channels using WebSockets on AWS to handle real-time sync with game engines like Godot.
- Gained a deeper understanding of Docker's isolation structure, learning to build lightweight images to optimize resource usage and accelerate deployment pipelines.
- Familiarized with automated deployment patterns and CI/CD pipelines—the golden key to transitioning from traditional Sysadmin to Cloud-DevOps engineering.

#### Career & Personal Growth
- Gained a transparent, practical career advancement roadmap derived from the speaker's first-hand experience, providing a solid blueprint for efficient self-study.
- Learned the "divide and conquer" approach and how to apply the Agile spirit to both personal task management and teamwork collaboration.

### Practical Applications

- **Implementing Docker:** Dockerizing the entire development environment of the current project to ensure absolute environment consistency between Local and Production.
- **Optimizing GraphRAG:** Researching the integration of AWS Neptune graph databases into our academic AI assistant to enhance its capacity for answering synthesized, highly relational queries.
- **Testing WebSockets:** Prototyping a minimal Proof of Concept (POC) utilizing serverless WebSockets to power real-time interactive features for current web/mobile applications.
- **Enhancing Infrastructure Security:** Proposing and configuring automated security layers by embedding Machine Learning models into our group's infrastructure firewall.
- **Boosting Teamwork Performance:** Promptly applying Agile communication techniques and sprint structures to eliminate overlapping task allocations within the team.
